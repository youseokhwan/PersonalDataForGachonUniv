2019.11.08(금) 10주차 Vue.js

[Vue.js]
- 배우기 쉽다.
- 리액트와 앵귤러에 비해 성능이 우수하고 빠르다.
- 데이터 바인딩 + 가상 돔(Virtual DOM)의 장점
  -> 변화된 부분만 변경해서 반영
- MVVM(View, ViewModel, Model) 패턴: 화면(View)과 데이터(Model) 사이를 연결하는 툴
- View
  DOM
  DOM Listener
  Vue.js는 화면단에서 데이터를 컨트롤하는게 가능
- 컴포넌트 기반 프레임워크
  1) 레고블록
  2) 리액트, 앵귤러도 마찬가지
  3) 코드의 재사용이 쉬움
  4) 화면의 구조를 직관적으로 판단하기 쉬움
- 특징
  1) 앵귤러의 양방향 데이터 바인딩
    - 화면(View)에 표시되는 값과 프레임워크의 모델(Model) 데이터 값이 동기화되어 한쪽이 변경되면 자동으로 변경됨
  2) 리액트의 단방향 데이터 바인딩
    - 데이터 전달 시 항상 상위 컴포넌트에서 하위 컴포넌트로 전달
