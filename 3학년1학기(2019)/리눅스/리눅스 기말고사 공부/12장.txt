--- 12장 Network & Telnet Server ---

0. Network 개요
- TCP/IP(Transmission Control Protocol / Internet Protocol) : 컴퓨터 간의 인터넷 통신 규약
- Gateway : 네트워크 간 정보를 전송하는 컴퓨터 또는 장비
- Router : 네트워크 간 경로를 지정해주는 장비
- Netmask : 네트워크의 규모를 결정
- 리눅스 네트워크 장치 이름 : ens33 (구 eth0, eth1)

- "nmtui" : Network Manager TUI. 텍스트 모드에서 네트워크 관리
- "rdate -s time.bora.net" : remote date. 타임서버의 시간 정보와 로컬을 동기화
- 국내 타임서버 리스트 NTP (Netowrk Time Protocol) : 무결성 인증

- /etc/sysconfig/network : 네트워크의 기본 정보가 담긴 파일
- /etc/sysconfig/network-scripts/ifcfg-ens33 : ens33의 정보 파일
- /etc/resolv.conf : DNS 서버 및 호스트 이름이 저장된 파일
- /etc/hosts : 현재 컴퓨터의 호스트 이름 및 FQDN(Fully Qualified Domain Name, www.abc.com처럼)이 있는 파일
- 리눅스는 ifconfig 사용(윈도우는 ipconfig)
- UUID(Universally Unique Identifier: 범용 고유 식별자)
  : 각각의 고유한 이름

Q. 표준 시간 3가지를 설명하시오
1) UTC(Universal Time Coordinated) : 세계 협정시
2) GMT(Greenwich Mean Time) : 그리니치 표준시
3) KST(Korean Standard Time) : 한국 표준시

Q. Telnet Server를 설치하는 과정을 서술하시오
1) yum install telnet-server
2) systemctl start telnet.socket
3) adduser teluser, passwd teluser
4) firewall-cmd --permanent --add-port=23/tcp
5) firewall-cmd --reload
6) systemctl restart telnet.socket (+ enable)
7) 클라이언트에서 서버의 ip로 telnet 접속
