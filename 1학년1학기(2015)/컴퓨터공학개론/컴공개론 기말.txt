음성인식 - 일반적으로 마이크나 전화를 통하여 얻어진 음향학적 신호를 단어나 단어 집합 또는 문장으로 변환하는 과정

음성인식의 특징
1. 숙련을 요하지 않는다.
2. 고속 입력이 가능하다.
3. 손발이 구속되지 않는다.

음성인식 - 컴퓨터가 음향학적 신호를 텍스트로 매핑시키는 과정
음성합성 - 문서를 입력하여 자동적으로 생성되는 합성 음성으로 변환하여 출력하는 것
음성이해 - 컴퓨터가 음향 음성 신호를 듣고서 음성의 의미로 매핑시키는 과정

UCC(User Created Contents) - 전문가가 아닌 일반인이 직접 제작한 콘텐츠

영상처리과정
1. 영상 데이터를 디지털로 획득하여 컴퓨터로 처리가 가능하도록 하는 일
2. 장치로부터 컴퓨터로 전송된 영상은 디지털 영상으로 전환
3. 영상의 결과는 출력장치를 통해 사용자가 알아볼 수 있도록 출력하거나 저장, 전송

ISDN - 음성, 데이터, 영상 등 다양한 형태의 정보를 하나의 통합된 망에서 제공하는 경제적인 디지털 통신망

컴퓨터 통신(네트워크) - 장치들을 그물 형태와 같이 서로 연결하여 S/W나 DB를 공유하도록 함으로써 효율적인 작업을 할 수 있도록 도와주는 환경

네트워크 구성평가 항목
1. 성능
2. 신뢰성
3. 보안

컴퓨터 통신 구성 요소
1. 송신자
2. 수신자
3. 메시지
4. 전송 매체
5. 프로토콜

전송 - 정보를 포함하는 신호를 전송 매체를 통해 공간의 임의의 한 점으로부터 다른 한 점으로 전달하여 주는 기능

기저 대역(baseband) 신호 - 다른 형태로 변환되지 않은 원래의 전기 신호

신호의 변조 - 신호를 전송, 수신할 수 있도록 신호의 형태, 주파수 범위를 변환하는 기능
다중화 - 여러 개의 신호를 하나의 신호로 묶어서 하나의 매체로 전송할 수 있도록 변환

아날로그 방식 - 연속적인 데이터
디지털 방식 - 이산적인 데이터

아날로그 신호를 디지털 신호로 변환
1. 표본화(샘플링)
2. 양자화
3. 부호화

전송의 기본 요소
1. 송신기
2. 전송로
3. 수신기

프로토콜 - 네트워크가 가능하도록 정해놓은 규칙

프로토콜의 구체적 기능
1. 주소 지정 - 상대방의 이름 지정
2. 순서 지정 - 데이터를 순서에 맞게 지정
3. 단편화와 재조합 - 데이터를 쪼개는 작업, 데이터를 합치는 작업
4. 흐름 제어 - 전송하는 데이터의 양, 속도 조절
5. 연결 제어 - 전송 시에 설정을 위한 구문, 의미, 시간을 제어하는 기능
6. 오류제어
7. 캡슐화 - 정확한 전송 및 데이터를 위하여 여러 가지 정보를 데이터에 추가
8. 동기화 - 특정 정보값을 공유하는 기능
9. 다중화 - 하나의 케이블에 여러 장치들이 동시에 통신할 수 있는 기능

토폴로지 - 통신망을 구성할 때 여러 가지 형태로 연결하여 구성하는 형태
1. 버스 토폴로지
2. 링 토폴로지
3. 스타 토폴로지
4. 트리 토폴로지 (제어간단, 관리확장 쉬움 / 중앙지점 고장 발생시 마비)
5. 메시 토폴로지
6. 애드혹 네트워크

데이터 통신망 - 데이터정보들의 정보 전달 체계

지점 간 회선 방식(point to point) - 1대1 연결
다지점 공유 회선(multi point) - 한개선에 여러개장치

연결방식에 따른 분류
1. 단방향 방식 (공중파 TV)
2. 반이중 방식 (무전기)
3. 전이중 방식 (전화기)

규모에 따른 분류
1. 근거리 통신망(LAN) - 한정된 지역의 네트워크
2. 원거리 통신망(WAN) - 광대한 영역의 네트워크
3. 대도시 통신망(MAN) - 근거리<대도시<원거리
백본 - 소형 데이터를 모아 빠르게 전송할 수있는 대규모 전송회선
4. 부가가치 통신망(VAN) - 독자적인 네트워크

전송 신호를 기준으로 분류
1. 베이스밴드 전송방식 - 디지털 데이터를 변조하지 않고 그대로 전송
2. 브로드밴드 전송방식 - 아날로그 신호로 변조한 후 전송

전송 매체의 접근 방식에 따른 분류
1. CSMA/CD - 회선을 감시하다가 회선이 비어있을경우 전송
2. 토큰링 - 고리 모, 전달 전달
3. 코드분할다중접속, 주파수분할다중접속, 시간분할다중접속

무선통신 - 전파를 전송 매체로 정보를 주고받는 방식

디지털 통신의 장점
1. 외부 신호의 방해에 대한 저항력이 크다

3. 소프트웨어에 의해 제어되는 유연성
4. 어떤 형태의 신호도 전송할 수 있는 창조성
5. 비용 및 전송 매체 이용의 효율성
6. 보안성

셀룰러 방식 - 주파수 사용 효율을 높이기 위해 한 셀에 이웃하고 있는 셀들에는 다른 중심 주파수를 배정하고, 셀 사이가 충분히 멀어 서로 통화 간섭을 주지 않는 거리에 있을 때는 기존의 배정된 주파수를 재활용

무선 전송 통신 수단

2. 지상 마이크로파
3. 위성 마이크로파

1. 블루투스 (10m 근거리, 2.4GHz)
2. Zigbee (50m, 2.4GHz)
3. RFID - 태그에 부착된 IC칩에 저장된 정보를 비접촉식 방법으로 인식
바코드는 빛, RFID는 전파
4. NFC - 비접촉식 근거리 무선통신

무선 근거리 통신(WLAN)
1. Wi-Fi
2. IEEE 802.11n

이동 통신 기술

인터넷 - 네트워크의 네트워크. 전세계에 퍼져있는 컴퓨터 간에 정보를 공유하기 위하여 컴퓨터와 통신망을 연결해놓은 최대의 컴퓨터 네트워크

TCP/IP - 인터넷의 기본적인 통신 프로토콜
인트라넷 - 기업 내에 속해있는 사설 네트워크

인터넷 주소

2. 도메인네임

IP주소 (32비트 체계), 부족해서 IPv6 개발 128비트

IP주소의 구성
1. Class A - 매우 큰 네트워크 0.0.0.0 ~ 127.0.0.0
2. Class B - 근거리 통신망과 호스트를 포함한 분산 처리 시스템에서 주로 사용
128.0.0.0 ~ 191.255.0.0
3. Class C - 일반적인 소규모 네트워크
192.0.0.0 ~ 223.255.255.0
4. Class D - 멀티클래스, 실생활 사용 X
24.0.0.0~239.255.255.0
5. Class E - 예비 목적용 혹은 실험용
익명연결 - 나머지 숫자가 0
브로드캐스트 - 나머지숫자가 255
private address
루프백 - 127.0.0.1 자신 가리킨다
