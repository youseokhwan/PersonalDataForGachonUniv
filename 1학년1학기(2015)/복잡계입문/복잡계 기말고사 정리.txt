복잡계 입문 기말고사

OX문제

1. 복잡계 패러다임의 특징은 작은 입력으로 막대한 효과, 비선형적, 인과관계, 순환고리적 상호관계, 시간의 흐름에 민감한 일시성을 갖는다. ( X )

2. 과학사를 중심으로 한 복잡계이론 발생 과정에서 가이아 이론이란 세계를 모든 현상의 상호 연관성과 상호 의존성에 의해 파악하는 이론을 말한다. ( X )

3. 프랙탈이론은 응용하면 설명이 불가능하리라 여겨졌던 복잡한 자연 형상을 단 몇 단계의 반복형 수식으로 나타낼 수 있다. ( O )

4. Floris Takens(타켄스)와 David Ruelle(루엘)은 난류와 기이한 끌개에 대해 바깥실린더는 정지되어 있고 안쪽 실린더는 회전하는 두 개의 실린더 사이의 유체 운동에 대하여 연구하였다. 그 결과는 회전 속도를 증가시키면서 실험했을 때, 유체의 운동은 프랙탈 차원을 가지는 기이한 끌개 모형을 형성하였다. ( X )

5. 심장마비 환자의 심박수와 정상인의 심박수의 Singularity Spectrum (특이성 스펙트럼)을 비교해보았을 경우, 심장마비 환자의 곡선이 더 좁게 나온다. ( O )

6. 프랙탈 예술의 4C는 Chaos(혼돈), Computer(컴퓨터), Creativity(창조성), Correspondence(조화)이다. ( X )

7. 시너제틱스의 인과고리가 작동하기 위해서는 시스템에 출입하는 에너지, 재화, 정보 등의 여러 환경요인이 맞아야 하며, 이들 조건은 제어매개변수로 나타내어진다. 적절한 제어매개변수 상태에서 시스템 요소들의 다양한 비선형 상호작용을 통해 규칙매개변수가 생성되고, 이들 규칙매개변수는 다시 시스템 요소들에게 영향을 미치며 개개 요소들의 다이내믹스를 지배한다. ( X )

8. 월프램(Wolfram)의 Class I & II는 랭턴(Lanton)의 결과에서 질서(order)에 해당하고 Class III는 복잡성(Complexity)에 해당하며, Class IV는 카오스(Chaos)에 해당한다. ( X )

9. 랜덤 네트워크(Random Network)의 대표적인 예시는 미국의 고속도로망이고 척도 없는 네트워크(Scale-free Network)의 대표적인 예시는 미국의 항공망이다. ( O )

10. 작은세상 네트워크(Small-World Network)는 특정한 노드만 장거리로 지름길을 가지고 있다. 따라서 소수 또는 일정 수의 장거리 연결자로 인해 전체 집단이 긴밀한 관계를 유지하는 상황에 적합하다. ( O )

11. 부분, 부분을 이해하며 조합하면 전체를 이해할 수 있다는 가정을 전제로 한 방법론은 환원주의이다. ( O )

서술형

1. 복잡계의 5가지 특징을 적으시오.

2. 자연과 일상생활 속에서 찾을 수 있는 프랙탈의 예시 5가지를 들으시오.

3. 나비효과의 예시를 찾아 서술하시오.

4. 다윈의 진화론과 카우만의 진화론의 차이점을 서술하시오.

5. 복잡계를 알아야 하는 이유와 실제로 사회와 문화 또는 예술에 활용되는 사례는 어떤 것이 있는지 서술하시오.

6. 복잡계의 5가지 이론은 무엇이 있으며, 각각의 이론의 개념과 특징을 서술하시오.

에세이

1. E. N. Lorentz(로렌츠) 모형의 3가지 결과를 적고 각 결과에 대한 적절한 예시와 설명을 적으시오.

2. 복잡계의 이론들 중 영화 `설국열차`에서 설명 될 수 있는 이론은 무엇이며, 그 이유는 무엇인가?

3. 복잡계의 이론들 중 영화 `I-Robot`에서 설명 될 수 있는 이론은 무엇이며, 그 이유는 무엇인가?
