인터넷 - 전 세계 모든 컴퓨터들을 연결하여 거리의 제한 없이 많은 정보를 교환할 수 있도록 만든 거대한 컴퓨터 통신망

기술적 측면 : TCP/IP 프로토콜을 기반으로 하는 네트워크
사회적 측면 : 네트워크를 이용하고 발전시켜 나가는 사람들의 공동체
정보적 측면 : 인터넷은 수많은 정보를 얻을 수 있는 자원의 집합
커뮤니케이션 측면 : 인터넷은 전 세계적인 네트워크

장점
1. 간편하고 빠르게 정보를 얻을 수 있음
2. 해당 분야의 전문가와 쉽게 의견을 교환할 수 있음
3. 자신의 정보를 다른 사용자에게 제공할 수 있음
4. 국가나 지역이 다른 사용자와도 공동 연구나 공동 작업을 진행할 수 있음
5. 전 세계의 정보를 제약 없이 얻을 수 있음

IPv4
1. IP 주소는 전 세계의 모든 컴퓨터들에게 부여된 고유의 식별 주소
2. 32비트로 구성
3. 32비트 전체를 8비트 단위로 분리하여 3개의 마침표로 구분하여 표현
4. 8비트로 구분된 단위는 옥텟 단위로, 255개 이하의 주소로 표현
5. 42억개의 주소 할당이 가능
6. 2011년 서비스 종료하고 IPv6으로 전환

클래스C
1. 소규모 네트워크환경
2. 네트워크 아이디는 24비트로 구성, 처음의 세비트는 110
3. 나머지 8비트는 호스트 아이디, 약 254개의 IP 주소를 사용할 수 있음

IPv6
1. IPv4의 대안
2. 128비트
3. 폭발적으로 증가하는 인터넷 사용에 대비할 수 있음
4. 속도 증가
5. 특정 패킷 인식을 통해 높은 품질의 서비스 제공
6. 보안성 강화
7. 사물 인터넷, 만물 인터넷, 유비쿼터스 통신 환경 구성

도메인 네임
1. 인터넷 환경에 존재하는 컴퓨터를 식별하기 위해 붙인 이름(호스트명)
2. 기억하기 쉬움
3. 최상위 도메인은 국가 표현

DNS(도메인 네임 서버) - 도메인 네임만으로 해당 호스트에 접속할 수 있도록 도메인 네임을 IP주소로 전환시켜 주는 역할 담당

서브넷마스크
1. 네트워크의 형태를 구분하는데 사용하는 주소
2. 서브넷마스크가 틀리면 인터넷 접속이 불가능

URL
1. 인터넷에서 사용하는 주소 표기 방법

TCP/IP
1. 인터넷 환경에서 데이터를 송수신하기 위해서 만들어진 표준 프로토콜

스위치
1. 하나의 LAN에 속한 컴퓨터끼리 통신이 가능하도록 네트워크를 구성
2. 라우터에 비해 빠르게 동작

브리지
1. 두 개의 근거리 통신망을 연결해주는 네트워크 연결 장치로 통신량 조절 가능

라우터
1. 인터넷 연결을 위한 필수 장비
2. 이기종 네트워크와 네트워크 사이의 데이터 전송을 위한 최적 경로 설정

게이트웨이
1. 2개 이상의 다른 종류 또는 같은 종류의 네트워크를 연결하여 통신망 사이의 정보를 교환할 수 있도록 하는 것

백본망
1. 소형의 근거리 통신망 회선들로부터 데이터를 수집해 빠르게 전송할 수 있도록 만든 대규모 전용회선, 기간망이라고함

웹
1. 다양한 정보를 하이퍼텍스트를 사용하여 방대한 DB를 구축하고 자유롭게 교환할 수 있게 만든 시스템

웹 2.0
1. 웹 2.0은 정보를 보여주는 역할에서 정보를 공유하고 함께 만드는 열린 플랫폼

텔넷
1. 인터넷을 통해 원격지에 위치한 호스트 컴퓨터에 접속할 때 사용하는 인터넷 표준 프로토콜

FTP (FIle Transfer Protocol)
1. 대량의 파일을 송수신하기위한 서비스
2. 웹보다 더 빠르게 전송 가능

포털 사이트
1. 인터넷을 이용하기 위해 기본으로 거쳐가는 웹사이트
2. 회원제, 이메일, 홈페이지 제공
3. 검색, 블로그 할 수 있음
4. 비즈니스의 토대
5. 모바일로 이동

SNS
1. 최근 가장 이슈가 되고 있는 커뮤니티
2. 페북, 트위터 등이 대표적
3. 주요 관심사에 대해 오랫동안 지속적으로 대화를 나누기 때문에 서로에 대해 잘 알게됨

모바일 미디어
1. 이동하는 중에도 무선 통신과 다양한 정보 처리를 할 수있는 기기

1세대 이동통신
1. 음성을 전송하는 아날로그 통신 방식 사용

2세대 이동통신
1. 음성을 디지털 신호로 변환
2. 문자 가능

3세대 이동통신
1. 영상통화, 멀티미디어 가능
2. 스마트폰 만들어지기시작함

4세대 이동통신
1. 고속 데이터 전송 가능
2. LTE는 4세대로 안봄
3. LTE-A

와이파이
1. 모바일 미디어 사이에 데이터를 직접 전송할 수 있도록 지원하는 표준 기술

블루투스
1. 휴대용 가전 기기 사이에서 근거리 무선 통신을 하기 위한 기술

NFC
1. 10cm 이내의 거리에서 디바이스끼리 데이터를 송수신 할 수 있는 기술
2. 기기를 맞대는 것만으로 기기 간 연동이 가능

아이비콘
1. 애플이 개발한 실내 위치 확인 시스템
2. 저전력 블루투스 장치인 비콘의 신호를 기반으로 고객의 위치를 탐지

커넥티즌
1. 연결과 주민의 복합어
2. 스마트폰과 같은 모바일 기기를 사용하여 업무를 즐기는 사람을 의미

LTE-A
1. 서로 다른 주파수 대역 2개를 합쳐서 사용

M-VolP
1. 모바일망을 이용한 데이터 통화 서비스

미래 인터넷
1. TCP/IP를 기반으로하는 단일 프로토콜 형태의 네트워크가 아니라 유무선, 방송, 인터넷 등과 같이 다양하고 이질적인 네트워크들이 하나로 연결되어 구성

미래 인터넷을 위한 조건
1. 접근성
2 개방성

기가 인터넷
1. 사용자에게 기가 이상의 전송 속도를 지원하는 차세대 인터넷
2. 고품질 대용량 콘텐츠를 고속으로 전송 가능

기가 와이파이
1. 60GHz 대역폭을 사용하여 데이터를 기가급으로 전송할 수 있는 기술

네트워크의 퀸텀 점프
1. 통신망의 속도가 한 단계 도약해야하는데 이를 `네트워크의 퀸텀 점프`라고함

사물인터넷(IoT)
1. 모든 사물이 인터넷으로 연결되어 스스로 정보를 생성, 수집, 공유하는 것을 뜻함

만물인터넷(IoE)
1. 세상 모든 만물이 인터넷으로 연결되어 소통하는 시대
2. 사물인터넷에서 확장된 개념

가상현실
1. 컴퓨터로 실제와 같은 환경을 만들고 그 속에서 사람의 오감을 통해 상호작용하면서 경험하는 것

HMD 디스플레이
1. 가상현실에서 몰입감과 입체감을 주기 위해 사용

오큘러스 리프트
1. 3D 가상현실 게임을 위한 헤드셋 장비

증강현실
1. 현실세계와 컴퓨터 영상을 합쳐 눈앞에 보여주는 기술

증강현실 문제점
1. 가상세계에 빠져 현실과 가상을 구분하지 못하는 상황 발생
2. 개인 정보 노출 문제
3. 스마트 안경을 끼고 사람들의 개인 정보를 얻는다면 심각한 문제 발생



14

디지털 콘텐츠
1. 아날로그 형태의 각종 정보를 0과 1로 디지털화하여 저장 및 가공 처리한 것

구성요소
1. 콘텐츠
2. 디자인
3. 정보통신기술

장점
1. 인터넷을 통해 유통과 배포를 빠르게 할 수 있음
2. 새로운 콘텐츠를 생성하기쉬움
3. 프로슈머, 상호작용 가능
4. 상호작용은 더 큰 부가가치 창출

단점
1. 시장이 협소
2. 개인정보유출, 무단 복제 문제
3. 시각과 청각에만 의존

번지 와치
1. 빠른시간에 여러 TV프로그램을 시청, 드라마나 만화를 1편부터끝까지 다보는 형태

비디오 커머스
1. 동영상으로 제품 정보를 제공

O2O 비즈니스
1. 온/오프라인의 각각 장점을 결합한 새로운 비즈니스모델
2. 사용자의 위치나 구매 이력등을 분석하거나 비콘 기기를 이용하여 근처에 있는 소비자의 스마트폰에 할인 쿠폰이나 이벤트 정보를 전송

OTT 서비스
1. 기존의 통신, 방송 사업자 외에 제 3의 사업자들이 온라인을 통해서 미디어 콘텐츠를 제공하는 서비스
2. 현재는 PC, 스마트폰 등의 단말기를 통해 제공되는 동영상 서비스와 모바일 결제 등을 포괄하는 의미로 사용

스트리밍 음원 서비스
1. 다운로드를 통해 원하는 음악을 소유하지않고 이어듣는 개념

해결과제
1. 저작권 사용로 문제
2. 망중립성 문제

바이럴 마케팅 광고
1. 블로그, 카페 등을 통해 사용자들이 자발적으로 제품을 홍보하는 방식

리마케팅 광고
1. 검색어를 바탕으로 웹사이트 방문자에게 해당 콘텐츠의 배너를 실시간으로 보여주는 마케팅 기법

리치미디어 광고
1. HTML5를 기반으로 미디어를 혼합하여 쌍방향으로 소통하는 멀티미디어 광고

모바일 리워드 광고
1. 스마트폰으로 광고를 보거나 앱을 다운받으면 보상해주는 광고 (돈버는 앱)

디지털 사이니지(Signage)개념
1. 디지털 디스플레이를 공공장소에 설치하여 각종 정보를 제공하는 제 4의 미디어
